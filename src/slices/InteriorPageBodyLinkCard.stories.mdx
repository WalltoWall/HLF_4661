# Link Card

Link to another page using a stylized card. Links can be internal or external.
A common use of this slice is to link to an external source article.

#### Slice fields

<Storybook.SliceFieldsTable
  customTypeId="news_post"
  sliceZoneId="body"
  sliceId="link_card"
  fieldset="non-repeat"
/>

### External link

If an external link (one that is not on the Omidyar Fellows site) is provided,
an "external link" icon will be displayed.

<Canvas withSource="none">
  <Story name="External link">
    <InteriorPageBodyLinkCard nextSharesBg={false} title={title} href="https://apple.com" />
  </Story>
</Canvas>

### Internal link

If an internal link (one that is on the Omidyar Fellows site) is provided,
an "external link" icon will *not* be displayed.

<Canvas withSource="none">
  <Story name="Internal link">
    <InteriorPageBodyLinkCard nextSharesBg={false} title={title} href="/" />
  </Story>
</Canvas>

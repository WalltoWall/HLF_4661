#### Interior Page

# Video

Displays a video from a third-party service such as YouTube or Vimeo. By
default, the cover image (a.k.a. poster image) from the third-party service will
be used. A higher-quality poster can be provided as an override.

#### Slice fields

<Storybook.SliceFieldsTable
  customTypeId="interior_page"
  sliceZoneId="body"
  sliceId="video"
  fieldset="non-repeat"
  overrides={{
    poster: {
      description: 'Cover image displayed before the video is played',
      isRequired: false,
    },
  }}
/>

### With poster override

The poster image provided to the Slice will be displayed before the user plays
the video. Providing an image allows you to use a higher quality image that the
default.

<Canvas withSource="none">
  <Story name="With poster override">
    <InteriorPageBodyVideo
      nextSharesBg={false}
      videoURL={videoURL}
      posterData={posterData}
    />
  </Story>
</Canvas>

### Without poster override

The default poster image from the third-party service will be used, but will
generally be of low quality.

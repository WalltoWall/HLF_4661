#### News Post

# Text Intro

Text block used to introduce content that follows. Basic formatting can be used
like bold text and links. Text in this Slice is automatically styled larger
than normal body content.

#### Slice fields

<Storybook.SliceFieldsTable
  customTypeId="news_post"
  sliceZoneId="body"
  sliceId="text"
  fieldset="non-repeat"
/>

### With a long introduction

Any amount of text can be used with this slice. The following example shows
this slice with a long introduction.

  <Story name="With a long introduction">
    <NewsPostBodyTextIntro nextSharesBg={false} textHTML={longTextHTML} />
